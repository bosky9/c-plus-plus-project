Since this project won't cover GAS models, the involved chunk of code has not been translated.

Markov blanket for each likelihood term - used for space state models.

Parameters
y Univariate time series
mean Array of location parameters for the Normal distribution
scale Scale parameter for the Normal distribution
Returns
Markov blanket of the Normal family

The original Python code is 

     return ss.norm.logpdf(y, loc=mean, scale=scale)

     Which returns the log of the probability density function at y of the given RV.
     Since mean can be a vector (in Python), the RV can be multi-dimensional, so, in order to translate this
     scipy.stats function,we implemented the Mvn class.

     The Mvn class (MultiVariate Normal) is needed to return samples obtained by a multi-dimensional Gaussian
     function.

     (Parameters shape and skewness were present but not actually used.)

◆ vi_loc_score()

template<typename T >
template double Normal::vi_loc_score< double > ( const T &  x ) const

Return the gradient of the location latent variable mu (used for variational inference) (Python code works with both a single float and a vector of floats)

Parameters
x A vector of random variables
Returns
The gradient of the location latent variable mu at x, for each variable

In Python, x could be a float or an array; if x was an array, the code would work anyway, using element-wise operations.

The template implementation allows x to be a double, or a dynamic Eigen:Vector of doubles. Both definition and specialization are included in the .cpp file.

Notice how, in order to implement element-wise operations, we need to cast x as an Eigen:Array: ... x.array() + 4 ... adds 4 to each element of x.
