BBVIReturnData BBVI::run ( bool  store )

The core BBVI routine - Draws Monte Carlo gradients and uses a stochastic optimizer.

Parameters
store If true, stores rgw history of updates for the benefit of a pretty animation
Returns
Results in BBVIReturnData structure

Using store, this function translates both "run" and "run_and_store". This "run" actually calls another function, run_with; this is to avoid troubles with the optional parameters of neg_posterior.
